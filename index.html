Antes de usar

Configurar el documento HTML

Bootstrap requiere el uso de HTML5:

<!doctype html>
<html lang= >
...
</html>

Al estar desarrollado mobile first, conviene el uso de la etiqueta meta viewport responsive dentro de <head>, asi los componentes se escalan de froma adecuada: 

<meta name= "viewport" content= "width=device-width, initial-scale=1" >

Configurar Bootrstarp

Es relevante saber que bootstrap cambia el valor global box-sizeing de content-box a border-box.

Hay varias formas de configurar Bootstrap para su uso:

  • Descargando de forma manual los Source Files
  • CDN (Contnent Delivery Network)
  • NPM (package manager)

Es necesiario jQuery para todos los widgets JavaScript

1. Source Files

Se descargan los archivos a utilizar desde la documentacion oficial de bootstrap y se colocan dentro de la carpeta del proyecto.

Para cargar los estilos CSS copiar y pegar lo siguiente dentro de <head> :

<link rel="stylesheet" href="css/bootstrap.css">

Para cargar los paquetes JavaScript copiar y pegar lo siguiente antes de cerrar la etiqueta <body>: 

<script src= "js/bootstrap.js" ></script>

Hay que instalar tambien Popper ya que alunos componentes de bootstrap dependen de este. Esto se puede hacer de dos formas:

  • NPM (package manager)
  • CDN (Contnent Delivery Network)

Hay una 3era opcion, en vez de cargar solo JS cambiarlo por el bundle que contiene Popper. Para esto hay que copiar y pegar lo siguiente antes de cerrar la etiqueta <body>:

<script src= "js/bootstrap.bundle.js" ></script>

2. CDN

Cargar CSS

Para cargar los estilos CSS copiar y pegar la siguiente hoja de estilo <link> donde suelen ir las hojas de estilo (dentro de <head>) :

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.2.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-Zenh87qX5JnK2Jl0vWa8Ck2rdkQ2Bzep5IDxbcnCeuOxjzrPF/et3URy9Bv1WTRi" crossorigin="anonymous">

Cargar paquete JavaScript (incluye Popper)

Para cargar copiar el siguiente <script> antes de cerrar la etiqueta <body> :

<script src= "https://cdn.jsdelivr.net/npm/bootstrap@5.2.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-OERcA2EqjJCMA+/3y+gxIOqMEjwtxJY7qPCqsdltbNJuaOe923+mo//f6V8Qbsw3"  crossorigin= "anonymous" ></script>

3. NPM

  1. Abrimos la terminal y entramos a la carpeta del proyecto.
  2. Escribimos "npm init", cuando nos pida un nombre colocamos "bootstrap-npm" y continuamos apretando enter (y salteando preguntas) hasta que se cree el archivo package.json .
  3. Installamos bootstrap: En la terminal escribimos "npm install bootstrap@5.2.2", lo cual va instalar todas las dependencias dentro de "node_modules".
  4. Linkeamos las dependencias al documento HTML con el siguiente codigo:

Para el CSS (en <head>) :

<link rel="stylesheet" href="node_modules/bootstrap/dist/css/bootstrap.css">

Para el JavaScript, (en <body> antes de cerrarlo):

<script src="node_modules/bootstrap/dist/js/bootstrap.bundle.js"></script>

Grid

Container

Lo primero que se necesita para poder trabajar con grid en bootstrap es un contenedor.

Ofrece varios tipos de contenedor:

  • .container --> ""deja margenes" ya que establece max-width a cada breakpoint establecido.
  • .container-fluid --> establece width: 100% por lo que ocupa el total de el ancho disponible.
  • .container-{breakpoint} --> que establece width: 100% hasta el breakpoint especificado.

Un ejemplo de como usarlos seria: 

<div class= "container-fluid"></div>

Creacion de filas y columnas

En bootstrap la creacion es similar a la de las tablas en HTML, es decir, se crea una fila ("row") y se crea cada una de las columnas ("col") que tiene esa fila.

<div class= "row">
    <div class= "col">Columna 1</div>
    <div class= "col">Columna 2</div>
</div>

Admite en cada fila hasta 12 columnas. Se le puede especificar un tamaño especifico a cada columna, especificando la cantidad de 1/12 que esta deberia ocupar. Por ejemplo:

<div class= "col-4">Columna 1</div>  /* Toma el tamaño de 4/12 */

En caso de querer que haya otra celda que ocupe las "porciones" faltantes, basta con crear una columna mas sin especificar su tamaño.

Se le pueden especificar breakpoints (incluso mas de uno en una misma columna). La sintaxis seria: col-breakpoint-tamaño. Esto permite que las columnas ocupen un espacio determinado dependiendo del viewport.

Alineacion

Las filas o columnas se pueden ubicar en el plano vertical respecto a los bordes del contenedor, de dos formas: aplicandole una clase a una columna especifica o a una fila (que se lo aplicaria a todas sus columnas). Estas son las clases:

En filas:

  • align-items-start
  • align-items-center
  • align-items-end

En las columnas inidviduales:

  • align-self-start
  • align-self-center
  • align-self-end

Tambien se pueden ubicar las filas en el plano horizontal respecto a los bordes del contenedor (similar a flexbox).

Estas son las clases:

  • justify-content-start
  • justify-content-center
  • justify-content-end
  • justify-content-around
  • justify-content-between
  • justify-content-evenly

Gutters

Columnas: Hacen referencia al espacio entre las columnas (el gap). La sintaxis de su clase seria gside-valor y se aplica en las filas ("row").

Donde los valores van de 0 a 5 y los sides son:

  • x --> los gaps verticales
  • y --> los gaps horizontales
  • --> gaps horizontales y verticales

Cards

Tamaño

Las cards vienen con un tamaño predefinido asignado en el <div> de la card bajo el atributo "style", en caso de eliminar este la card toma el mayor ancho disponible.

Grid

Se puede determinar su tamaño de forma sencilla usando grid, los elementos (la card) se colocan dentro de una columna del grid de la siguiente forma:

<div class= "row">
                    <div class= "col">
                    <div class= "card"> /* Se le eleimino el atributo "style" que le daba su estilo predefinido */
                    /* Selementos de la card */   
                    </div>
                    </div>
</div>

Usando Utilidades

Se pueden usar las utilidades de tamaño de bootstrap, las cuales asumen que no tiene ancho especificado y permite asignarle 25%, 50%, 75% y 100% del ancho total, con la siguiente clase: w-porcentaje. Esta se coloca en el <div> de la card.

Usando CSS personalizado

Se le cambian los valores al atributo "style" que se encuentra dentro de la <div> de la card. alli colocamos o modificamos los valores de width: a nuestro gusto.

Encabezado y pie de pagina

Para agregarle a la card ya hecha un encabezado hay dos formas:

  • se le agrega un <div> arriba del todo (dentro del <div> de la card) que contenga la clase "card-header", y dentro de este colocamos el texto del encabezado.
  • se le agrega la clase "card-header" a un elemento <h>(un titulo).

Para agregarle a la card ya hecha un pie de pagina basta con hacer un <div> al final del <div> de la card, que contenga la clase "card-header", y dentro de este colocamos el texto del pie de pagina. 

<div class=  "card">
<div class=card-header>
/* Iria el texto de encabezado */
<div>
                    <div class=  "card-body">
                    /* Aca iria algun texto */
                    <div>
<div class=card-footer>
/* Iria el texto de pie de pagina */
<div>
<div>

Horizontales

Bootstrap permite hacer cards horizontales responsive, que pueden convertirse en verticales dependiendo del tamaño del viewport. Para esto hace uso de grid, donde por ejemplo en una columna se coloca la imagen y en otra el body de la card. Es necesario agregar la clase g-0 a la fila ("row") para que esto funcione.

Para hacerlo responsive y que se haga vertical basta con especificar el breakpoint en la columna ("col").

Disposición de las Cards

Grupo de Cards

Es util para crear un grupo de cards que tengan igual altura y ancho, y esten contenidas en un unico elemento. Cabe aclarar que no es responsive (Existe otra forma que si es responsive de hacer esto que seria usando grid en la card como se hace con las cards horizontales pero con mas columnas)

Para lograr esto basta con contener cada card dentro de un <div> que tenga la clase "card-group".

Estilos de Cards

Color de fondo y texto

Para cambiar el color de fondo y/o texto de las cards se les puede agregar alguna de las siguientes clases:

  • text-colorPredefinido --> Establece el color del texto al colorPredefinido.
  • bg-colorPredefinido --> Establece el color de fondo al colorPredefinido.

Bordes

Se le puede cambiar el color de borde a la card con la siguiente clase: "border-color"

Tipografía

Títulos

De forma preestablecida se usan los titulos <h1> a <h6> de Bootstrap.

Ademas se pueden usar las clases: .h1 a .h6 si se quiere que un texto se vea como titulo pero no aparezxca asi en el DOM.

Todos estos son responsive.|

Títulos Display

Titulos mas grandes, con las clases: display-número, yendo los numeros del 1 al 6.

Elementos de texto

Para usar de forma semantica tenemos los siguientes:
  • Highlight --> <mark>texto</mark>
  • Eliminado --> <del>texto</del>
  • Tachado --> <s>texto</s>
  • Complemento --> <ins>texto</ins>
  • Subrayado --> <u>texto</u>
  • Letra chica --> <small>texto</small>
  • Negrita --> <strong>texto</strong>
  • Cursiva --> <em>texto</em>

En caso de buscar estilizar usar las siguientes clases:

  • Highlight --> .mark
  • Letra chica --> .small
  • Subrayado --> .text-decoration-undeerline
  • Tachado --> .text-decoration-through

Blockquotes

Para citar bloques de contenido de otras fuentes:

<blockquote class= "blockquote">
                    /* Aqui iria una quote */
</blockquote>

En caso de querer nombrar la fuente, HTML requiere que se haga afuera de <blockquote> y dentro de <figcaption>. Por lo que se envuelven en el elemento <figure> de la siguiente forma:

<figure>
                    <blockquote class= "blockquote">
                    /* Aqui iria una quote */
                    </blockquote>
                    <figcaption class="blockquote-footer">
                     /* La fuente de la quote */
                    </figcaption>
</figure>
